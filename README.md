# Bot-telegram

> unofficial telegram api wrapper php

![Swag logo](assets/header.png)

## Installation

```sh
composer install
```

## Overviews

```php
require __DIR__ . '/vendor/autoload.php';
use Telegram\Bot\Client;
$bot = new Client('BOT_TOKEN');
$bot->sendMessage('hello world','1234'); // message, to
```
